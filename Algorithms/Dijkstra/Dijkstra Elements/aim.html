Aim

The aim of this virtual lab is to understand the concept and implementation of Dijkstra algorithm.

Estimated Time

1 hour

Learning Objectives of the Experiment

  • Learn about the Hashing technique and how it can be used to solve search and indexing problems.
  • Given a dataset of keys, generate hash values and store them in a hash table using an appropriate hashing function.
  • Understand Hashing operations: insertion, deletion, and searching, as well as collision resolution techniques like chaining and open addressing, through interactive animations.
  • Demonstrate knowledge of time complexity of Hashing by analyzing the performance of different hashing functions and collision resolution strategies.
  • Compare Hashing with other search algorithms and realize Hashing as an efficient technique for quick data retrieval.
