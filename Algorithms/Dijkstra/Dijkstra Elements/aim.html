Aim

The aim of this virtual lab is to understand the concept and implementation of Dijkstra algorithm.

Estimated Time

1 hour

Learning Objectives of the Experiment

  • Learn about Dijkstra's Algorithm and how it can be used to solve shortest path problems in weighted graphs.
  • Given a dataset of nodes and edges, implement Dijkstra's Algorithm to calculate the shortest paths from a starting node to all other nodes.
  • Understand Dijkstra's Algorithm operations: initialization, selecting the minimum distance node, updating distances, and final path retrieval, through interactive animations.
  • Demonstrate knowledge of time complexity of Dijkstra's Algorithm by analyzing its performance with different data structures (e.g., priority queues) and graph types.
